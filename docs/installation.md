# Installation

## Stable release

To install GeneralDSML, run this command in your terminal:

```sh
uv add GeneralDSML
```

Or if you prefer to use `pip`:

```sh
pip install GeneralDSML
```

## From source

The source files for GeneralDSML can be downloaded from the [Github repo](https://github.com/druiz35/GeneralDSML).

You can either clone the public repository:

```sh
git clone git://github.com/druiz35/GeneralDSML
```

Or download the [tarball](https://github.com/druiz35/GeneralDSML/tarball/master):

```sh
curl -OJL https://github.com/druiz35/GeneralDSML/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd GeneralDSML
uv pip install .
```
